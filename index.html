Drop it into any page

Two files, no build step. Link the stylesheet, opt your <body> into Nyx, and add the runtime for tabs, modals and toasts.

Read the guide 
<!-- 1. the framework -->
<link rel="stylesheet" href="nyx.css">

<!-- 2. opt the body in -->
<body class="nyx nyx-reset">
  <button class="nyx-btn nyx-btn-primary">Hello</button>

<!-- 3. the runtime -->
<script src="nyx.js"></script>
Why Nyx

Designed for the SaaS era

Opinionated defaults, a token system you can retheme in one line, and behaviors that need no JavaScript to wire up.

✦

Luminous Depth

Every interactive element is lit from within — colored glows, layered surfaces, glass.

⚡

Zero dependencies

Two files, ~500 lines of CSS, a tiny vanilla-JS runtime. Just Google Fonts.

🎛️

Declarative JS

Modals, tabs, drawers and scrollspy wire up with data-nyx-* — no code.

🎨

Token theming

Override any --nyx-* variable to retheme everything. No recompile.

🌙

Dark-mode native

Built dark-first, the way modern SaaS tools actually look.

📱

Fully responsive

Works from a 1440px dashboard down to a 375px phone. Accessible by default.
